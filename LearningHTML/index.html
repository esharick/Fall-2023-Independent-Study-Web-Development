Displaying text

The <p> tag can be used for other text.
<br> adds a line break between elements.

<hr> is a horizontal divider

<pre> is used for preformatted text.
Or use 'code' to show code.
x = 3; y = 5;
print(x + y);

You can style elements with different tags: bold, strong, italic, emphasized, mark, small, deleted, inserted, subscript, superscript.

You can use the 'style="color:blue"' attribute to color elements, but you should keep styles in a style sheet (CSS)

I'll show more styles on the CSS page.

Navigation and Links

The 'a' tag can create a link element. It can link between pages or elements on a page.
You can also use text, images, buttons, and other elements as links.
Just set the href="url" and set the target="_self/_blank/_parent/_top".

Tables

The tags are 'table', 'tr' (table row), 'th' and 'td'
