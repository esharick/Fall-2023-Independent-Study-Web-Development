<br> adds a line break between elements.

<hr> is a horizontal divider

Or use 'code' to show code.

Navigation and Links

The 'a' tag can create a link element. It can link between pages or elements on a page.
You can also use text, images, buttons, and other elements as links.
Just set the href="url" and set the target="_self/_blank/_parent/_top".
